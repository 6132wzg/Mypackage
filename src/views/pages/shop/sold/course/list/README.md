list新增需求
筛选字段： 课程包类型 （限次，不限次）

列表字段： 1. 课程包类型
              限定次数约课
              不限次数约课
          2，属性
            任意课程
            范围课程
            固定课程
          3. 剩余次数和购买次数
             1. 团课
             2. 团课/私教课
             3. 共X次

info 新增需求

1. 剩余课时混合次需要修改
2. 去掉节数，课时费、小计，新增扣次额度，平铺

### 操作新增和修改
  1. 升级： 根据选的不同课程包种类选择填写不同的额度
      仅支持团课，仅支持私教 或者团课和私教
  2. 转让，前端不需要修改，但是列表项会增加 团课&私教课
  3. 扣次： 同升级需求类似 对课程包不同约课额度进行扣次
  4. 修改剩余课时，前端不需要修改 同转让
